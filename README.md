# 🦞 OpenClaw Setup Guide

AI assistant bot connected to WhatsApp, Telegram, Discord and more — powered by Claude AI on WSL Ubuntu.

## What is OpenClaw?
OpenClaw is an open-source AI gateway that connects Claude AI to messaging platforms like WhatsApp, Telegram, Discord, and more.

## Requirements
- Windows with WSL2 (Ubuntu 24)
- Node.js v18+
- npm v10+
- Claude CLI installed and authenticated

## Channels Status
- [x] WhatsApp
- [ ] Telegram
- [ ] Discord
- [ ] Microsoft Teams

## Quick Start
```bash
curl -fsSL https://openclaw.ai/install.sh | bash
openclaw onboard --install-daemon
```
